# Label Editor
이미지 어노테이션 도구 · COCO dataset 형식 호환

---

## 도구

| 키 | 도구 | 설명 |
|---|---|---|
| `D` | 폴리곤 그리기 | 꼭짓점을 클릭해 다각형을 그림. 첫 점 근처 클릭 또는 더블클릭으로 닫기 |
| `B` | 브러시 | 좌클릭으로 칠하고 우클릭으로 지움. `[ ]`로 크기 조절 |
| `M` | AI 매직완드 | 클릭 한 번으로 영역 자동 인식 (EdgeSAM). `[ ]`로 마스크 후보 선택 |
| `H` | Pan | 이미지 이동. 한 번 더 누르면 이전 도구로 복귀 |

---

## 기본 흐름

**폴더 열기** `Ctrl+O` → **클래스 추가** → **도구 선택** `D / B / M` → **마스크 확정** `Enter` → **저장** `Ctrl+S`

---

## 레이블 편집

- 우측 **Labels** 패널에서 레이블 클릭 → 편집 모드 진입
- 꼭짓점(노란 점)을 드래그해 윤곽선 조정 — 브러시 모드에서는 비활성화
- `B` + 브러시 페인팅으로 마스크 직접 수정
- `Esc` 로 편집 모드 종료

---

## 단축키

| 키 | 기능 |
|---|---|
| `D` | 폴리곤 그리기 |
| `B` | 브러시 |
| `M` | AI 매직완드 |
| `H` | Pan · 한 번 더 누르면 이전 도구 복귀 |
| `Space` | 누르는 동안 임시 Pan |
| `[ / ]` | 브러시 크기 조절 · 매직완드 마스크 후보 선택 |
| `Enter` | 마스크 확정 |
| `Esc` | 취소 · 편집 모드 종료 |
| `Ctrl+Z` | 실행 취소 |
| `Ctrl+S` | 저장 |
| `Ctrl+O` | 폴더 열기 |
| `F` | 이미지 맞춤 (Fit) |
| `= / -` | 확대 / 축소 |
| `V` | 레이블 흐리게 보기 토글 |
| `G` | 감마 보정 토글 |
| `Delete` | 선택된 레이블 삭제 (Labels 패널 포커스 중) |

---

## 저장

- 이미지당 하나의 `.json` 파일 · LabelMe 형식 호환
- 어노테이션이 없는 이미지는 JSON 파일을 생성하지 않음
- 이미지 목록의 📄 아이콘 — 해당 이미지의 JSON 존재 여부 표시
- 종료 또는 새 폴더 열 때 미저장 변경사항이 있으면 **Save / Discard / Cancel** 창 표시
